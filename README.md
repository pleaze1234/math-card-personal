# Math Cycle

개인용 온디바이스 수학 트레이너입니다. PDF를 휴대폰에서 넣고, 문제/해설 이미지를 자동 카드화한 뒤 학습 사이클로 반복 풀이합니다.

## v5 변경 사항

- 별점 기능 전체 삭제
- 메모 기능 삭제
- 드로잉은 해설 보기 전부터 활성화
- 메인 UI를 `학습 사이클 / 문제집 관리` 상단 탭 구조로 변경
- 학습 사이클 카드에 이름, 문제집 수, 출제 순서, 진행도, 문항 수 표시
- 사이클 카드에 `계속 학습 / 오답 문항 / 삭제` 버튼 추가
- 맞음/틀림 버튼은 상태만 변경하고, 다음 문제 이동은 하단 `다음 문제` 버튼으로만 수행
- Android 휴대폰 뒤로가기 버튼 처리 추가
  - 풀이 화면 → 사이클 또는 카드 화면
  - 카드 화면 → 문제집 관리
  - 오답 목록 → 학습 사이클
  - 메인에서 뒤로가기 → 종료 확인창
- PDF 파싱 보강
  - `【1】 정답` 형식
  - `1 【】정답` 형식
  - 줄 중간에 독립적으로 들어간 `【6】` 같은 해설 표지 후보 추가 감지
  - 문제 페이지에서는 해설 탐지 차단 유지

## GitHub Actions 빌드

저장소 최상단에 아래 파일들이 있어야 합니다.

```text
.github/
src/
package.json
capacitor.config.ts
tsconfig.json
index.html
README.md
```

GitHub에서 `Actions → Build Android APK → Run workflow`를 실행하면 `math-card-debug-apk` artifact가 생성됩니다.

## 사용 흐름

1. `문제집 관리` 탭에서 PDF 넣기
2. 파싱 완료 후 문제집 선택
3. 개별 문항 확인 또는 문제 풀이
4. `학습 사이클` 탭에서 사이클 제작
5. 풀이 중 해설보기 전에도 드로잉 가능
6. 해설을 본 뒤 맞음/틀림 상태 선택
7. 하단 `다음 문제`로 이동
8. 사이클 카드의 `오답 문항`에서 오답만 확인


## v6 수정

- 오답 문항 목록에서 문항을 열면 이전/다음 이동이 해당 사이클의 오답 문항 안에서만 동작합니다.
- 오답 문항 화면이나 오답 문항에서 열린 카드/풀이 화면에서 뒤로가기를 누르면 다시 오답 문항 목록으로 돌아갑니다.
- PDF 분석 중에는 상세 로그 대신 진행도만 표시합니다.
- 선형대수 PDF처럼 해설 시작 표지가 `1` + `【】정답`처럼 서로 다른 텍스트 조각으로 분리되어 추출되는 경우를 추가 인식합니다.
